## スライド 4: 主要機能

### 1. 書類作成の自動化

| 対応書類 | 自動化内容 |
|----------|-----------|
| 工事写真台帳 | 写真から自動生成 |
| 施工報告書 | 進捗データから作成 |
| 日報 | 作業記録を自動集計 |
| 検査報告書 | テンプレート＋AI補完 |

**効果**: 書類作成時間 **70%削減**

---

### 2. 写真管理AI

- **自動分類**: 撮影した写真を工程別に自動振り分け
- **タグ付け**: AI画像認識による自動タグ生成
- **検索**: 自然言語で写真を瞬時に検索

```
撮影 → 自動分類 → タグ付け → クラウド保存
  ↓
「杭打ち工程の写真を探して」
  ↓
AI検索で即座に表示
```
